PYANdex game
============

PYANdex - карточная игра "пьяница" для компании yandex

HowTo
-----

    # server tests run
    cd server
    npm install
    npm start

    cd ..

    # browser run
    cd client
    bower install
    cd ..
    python -m SimpleHTTPServer 8000
    open http://172.0.0.1:8000/client/
